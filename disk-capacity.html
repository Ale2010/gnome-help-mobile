Check how much disk space is left

You can check how much disk space is left by using Disk Usage Analyzer and System Monitor.

Check with Disk Usage Analyzer

To check the free disk space and disk capacity using Disk Usage Analyzer:

  1. Open the Disk Usage Analyzer application from the Activities overview.

  2. The Disk Usage Analyzer window will display the Total file system capacity and Total file system usage.

    (You can click on View as Rings Chart to change the display to View as Treemap Chart).

  3. Click Scan Home if you want to check how much disk space is left inside the Home folder or click Scan filesystem icon if you want to check how much disk space is left inside the '/' (Root) folder. In both cases, the location of the files and other options like Usage, Size and Contents will be presented. You can change the order of the details and view the files in order of file size.

  4. Click the Scan a folder icon if you want to scan disk space inside specific folder or click Scan a remote folder if you want to scan disk space inside remote folder. See more details in Disk Usage Analyzer

  5. Click Stop or Refresh anytime you need to cancel or update the scanning.

Disk Usage Analyzer can also be run by pressing Alt+F2, typing 'baobab' (without the single-quotes) and pressing Enter.

Check with System Monitor

To check the free disk space and disk capacity with System Monitor:

  1. Open the System Monitor application from the Activities overview.

  2. Select the File Systems tab to view the system's partitions and disk space usage in Total, Free, Available and Used.

What if the disk is too full?

If the disk is too full you should:

  • Delete files that aren't important or that you won't use anymore.

  • Make backups of the important files that you won't need for a while and delete them from the hard drive.
