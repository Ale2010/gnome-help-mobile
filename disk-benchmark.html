Test the performance of your hard disk

If you want to perform benchmark test and see how fast is your hard disk:

  1. Open the Disk Utility application from the Activities overview.

  2. You can choose the hard disk from the Storage Devices list. Information about the disk and its status will appear under Drive.

  3. Choose Benchmark.

  4. Click Start Read-Only Benchmark to test how fast data can be read off the disk or Start Read/Write Benchmark to test how fast data can be read off the disk and how fast data can be put onto the disk. The last option will take longer to complete.

When the test is finished, the results will appear on the graph.
