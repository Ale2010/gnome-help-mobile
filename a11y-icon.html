Find the universal access menu

The universal access menu is where you can turn on various accessibility settings. You can find this menu by clicking the icon which looks like a person surrounded by a circle on the top bar.

The universal access menu can be found on the top bar.

To access this menu using the keyboard rather than the mouse, press Ctrl+Alt+Tab to move the keyboard focus to the top bar. A white line will appear underneath the Activities button - this tells you which item on the top bar is selected. Use the arrow keys on the keyboard to move the white line under the universal access menu icon and then press Enter to open it. You can use the up and down arrow keys to select items in the menu. Press Enter to toggle the selected item.
