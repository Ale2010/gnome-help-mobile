Pin your favorite apps to the dash

To add an application to the dash for easy access:

  1. Open the Activities overview.

  2. Find the application you want to add to the dash by clicking on Applications. Right-click on the application icon and select Add to Favorites.

    Alternately, you can click-and-drag the icon into the dash.

To remove an application icon from the dash, right click on the application icon and select Remove from Favorites. You can also drag the application icon to the Trash icon on the dash to remove it.
