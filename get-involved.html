Participate to improve this guide.

This help system is created by a volunteer community. You are welcome to participate. If you notice a problem with these help pages (like typos, incorrect instructions or topics that should be covered but aren't), you can file a bug report. To file a bug, go to bugzilla.gnome.org.

You need to register so you can file a bug and receive updates by e-mail about its status. If you don't already have an account, just click on the New link to create one.

Once you have an account, log in, click on File a Bug ▸ Desktop ▸ gnome-user-docs. Before reporting a bug, please read the bug writing guidelines, and please browse for the bug to see if something similar already exists.

To file your bug, choose the component in the Component menu. If you are not sure which component your bug pertains to, choose general.

If you are requesting help about a topic that you feel is not covered, choose enhancement in the Severity menu. Fill in the Summary and Description sections and click Commit.

Your report will be given an ID number, and its status will be updated as it is being dealt with. Thanks for helping make the GNOME Help better!
