Move a window to a different workspace

Using the mouse:

  1. Open the Activities overview and make sure you're looking at the Windows view.

  2. Click and drag the window toward the right of the screen.

  3. The workspace selector will appear.

  4. Drop the window onto an empty workspace. This workspace now contains the window you dropped into it, and a new empty workspace appears at the bottom of the workspace panel.

Using the keyboard:

  1. Click on the window to make it live.

  2. Press Ctrl+Alt+Shift+↑ to move the window to a workspace which is above the current workspace the workspace selector.

  3. Press Ctrl+Alt+Shift+ ↓ to move the window to a workspace which is below the current workspace the workspace selector.
