Turn on Bounce Keys

Turn on Bounce Keys to ignore keypresses that are rapidly repeated. For example, if you have hand tremors which cause you to press a key multiple times when you only want to press it once, you should turn on bounce keys.

  1. Click your name on the top bar and select System Settings.

  2. Open Universal Access and select the Typing tab.

  3. Select Turn on accessibility features from the keyboard.

  4. Switch Bounce Keys on.

Use the Acceptance delay slider to change how long bounce keys waits before it registers another key press after you pressed the key for the first time. Select Beep when a key is rejected if you want the computer to make a sound each time it ignores a key press because it happened too soon after the previous key press.

You can quickly turn Bounce Keys on and off by clicking the accessibility icon on the top bar and selecting Bounce Keys.
