Switch between workspaces

Using the mouse:

In the Activities overview, click on a workspace in the workspace selector on the right side of the screen to view the open windows on that workspace. Click on any window thumbnail to activate the workspace.

Using the keyboard:

  • Press Ctrl+Alt+↑ to move to a workspace which is above the current workspace in the workspace selector.

  • Press Ctrl+Alt+↓ to move to a workspace which is below the current workspace in the workspace selector.
