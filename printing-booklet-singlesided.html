Print a booklet on a single-sided printer

These instruction are for printing a booklet from a PDF document. If you want to print a booklet from a LibreOffice document, first export it to a PDF. Make sure the document has a multiple of 4 pages. You may need to add up to 3 blank pages.

To print:

  1. Click File ▸ Print.

  2. Choose the General tab.

    Under Range, choose Pages.

  3. Type the numbers of the pages in this order (n is the total number of pages, and a multiple of 4):

    n, 1, 2, n-1, n-2, 3, 4, n-3, n-4, 5, 6, n-5, n-6, 7, 8, n-7, n-8, 9, 10, n-9, n-10, 11, 12, n-11...

    ...until you have typed all of the pages.

    Examples:

    4 page booklet: Type 4,1,2,5

    8 page booklet: Type 8,1,2,7,6,3,4,5

    12 page booklet: Type 12,1,2,11,10,3,4,9,8,5,6,7

    16 page booklet: Type 16,1,2,15,14,3,4,13,12,5,6,11,10,7,8,9

    20 page booklet: Type 20,1,2,19,18,3,4,17,16,5,6,15,14,7,8,13,12,9,10,11

  4. Choose the Page Setup tab.

    Under Layout, in the Two-side menu, select One Sided.

    In the Pages per side menu, select 2.

    In the Page ordering menu, select Left to right.

    In the Only print menu, select Odd sheets.

  5. Click Print.

  6. When all the pages have printed, flip the pages over and place them back in the printer.

  7. Click File ▸ Print.

  8. Choose the Page Setup tab.

    In the Only print menu, select Even sheets.

  9. Click Print.
