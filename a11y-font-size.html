Change text size on the screen

If you have difficulty reading the text on your screen, you can change the size of the font.

  1. Click your name on the top bar and select System Settings.

  2. Open Universal Access and select the Seeing tab.

  3. Under Display select the Text size that is big enough for you. It will adjust immediately.

You can quickly change the text size by clicking the accessibility icon on the top bar and selecting Large Text.
